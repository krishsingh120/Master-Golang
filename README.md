# Master GoLang

## Reason to choose golang

1. Build time
2. fast startup
3. performance and efficiency
4. concurrency model
5. static type and compilation


## Introduction

### 1. Package main is standard lib in go
### 2. All packages inside go are came from main package

### Build go program
```bash

## execute build file
./<file_name>
```

### Direct run go program
