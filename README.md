# Master GoLang

## Reason to choose golang

1. Build time
2. fast startup
3. performance and efficiency
4. concurrency model
5. static type and compilation

### 1. Package main is standard lib in go

### Build go program
```bash
go build <file_path>
## execute build file
./<file_name>
```

### Direct run go program
```bash
go run <file_path>
```
